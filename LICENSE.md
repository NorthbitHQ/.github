# NorthbitHQ Organization-Wide Licensing Policy

This repository governs the overall licensing policies for the NorthbitHQ GitHub organization.

## How Licensing Works in Our Organization

Each repository within the NorthbitHQ organization contains a `LICENSE` file that specifies the exact terms and conditions applicable to its codebase.

To see the portfolio of licenses we use and to understand their terms, please refer to our **[LICENSE-LIST.md](./LICENSE-LIST.md)**.

That document provides a clear breakdown of each license type, its use case, and a link to the full license text.

## Contributions
